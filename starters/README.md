# ZizkaDB starters

Same templates as `zizkadb init` — kept in-repo for GitHub discovery and forks.

```bash
pip install zizkadb-sdk
zizkadb init my-bot -t basic
```

Or copy a folder from `sdk/python/zizkadb/templates/`.

Templates: **basic** · **openai** · **langchain** · **crewai** · **mcp-cursor**
